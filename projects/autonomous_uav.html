Autonomous UAV

Personal Project  •  2024 – 2025

This project started as a passion for building and flying multirotors and evolved into a complete autonomous system. I designed the airframe, integrated sensors and wrote flight control software that allows the quadrotor to navigate through waypoints without manual input. The experience taught me about sensor fusion, state estimation and low‑level embedded programming.

Key Features

  • Custom airframe with 3D‑printed mounts for flight controller and sensors
  • Onboard perception using an IMU and GPS for state estimation
  • PID attitude controller implemented in C
  • Waypoint navigation using a finite state machine
